# Nancy Report — June 16

Web-based presentation built from the original Google Slides.

**Live:** https://jeanine23.github.io/nancy-report-june-16/

## Usage

- Arrow keys, Space, or click left/right edges to navigate
- URL hash routes to slides: `#1`, `#2`, ... `#19`
- Click "Download .pptx" to grab the editable file

## Files

- `index.html`, `styles.css`, `script.js` — the deck
- `slides/slide_NN.png` — rendered slide images (1920×1080)
- `Nancy-Report-June-16.pptx` — original editable source
